# Getting Started

## Intel Mac

- Use vertualenv
- Required Python 3.7

```
pip install tensorflow==2.4.0
python mnist_convnet.py
```

## M1 Mac

- Clone [tensorflow_mac](https://github.com/apple/tensorflow_macos), and run script at local.
  - including...
    - Python 3.8
    - Virtualenv
    - Tensorflow

```
python mnist_convnet.py
```
